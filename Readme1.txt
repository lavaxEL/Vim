Для начала работы с Vim следует запустить Vim.exe
Стартовый режим работы Vim - normal(режим навигации и ввода команд)
Программа работает в следующих режимах:
          - normal mode - основной режим (большинство команд и переход в остальные режимы осуществляются именно через этот режим)
          - input mode - режим редактирования текста
          - command mode - режим сохранения/открытия файла
          - hex mode - режим hex-редактора
1) Normal mode. Основной режим работы. В данный режим можно вернутся из любого режима с помощью клавиши "Esc".
!!!Доступные команды можно посмотреть на правой панели после ввода "h" в командную строку.
2) Input mode. Режим редактирования текста. Доступен ввод текста без открытия файла (команда "i"). Сохранения файла 
возможно впоследствии через command mode. Другой вариант сначала открыть файл, а затем перейти в его редактирование.
3) Command mode. Режим для открытия файлов и сохранения. 
!!!Доступные команды можно посмотреть на правой панели после ввода "h" в командную строку.
4) Hex mode. Режим hex-редактора. Доступна одна комнда:
          - "NG", где N - любое число из диапазона 0-FFFF. Команда выполняет перевод курсора да данный байт.

Навигация (перемещение курсора) во всех режимах работы Vim производится с помощью стрелочек.

P.S. В режиме Input mode поддерживаются клавиши Del, Bs и Enter. Выполняют функционал схожий с функционалом
других текстовых редакторов.
P.P.S. В качестве тестового файла - test.txt
